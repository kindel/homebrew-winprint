# WinPrint Homebrew tap

Free **WinPrint TUI** (`wp`) for macOS and Linux.

```sh
brew install kindel/winprint/winprint
wp --version
```

`Formula/winprint.rb` is updated automatically by the release pipeline in
[kindel/winprint](https://github.com/kindel/winprint) on each stable release
(`.github/workflows/release.yml` → `brew` job). Do not edit it by hand.

> The WinPrint **GUI** is distributed separately (Microsoft Store / Mac App Store), not via this tap.
